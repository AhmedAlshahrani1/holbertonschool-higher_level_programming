# SQL - Introduction

This project is an introduction to relational databases and MySQL.

## Description

Basic SQL scripts covering: creating and deleting databases, creating and
altering tables, and performing CRUD operations (SELECT, INSERT, UPDATE,
DELETE) using MySQL 8.0 on Ubuntu.

## Requirements

- All files are executed on Ubuntu 22.04 LTS using MySQL 8.0
- Each file starts with a comment describing the task
- All SQL keywords are written in uppercase
