# Python - Classes and Objects

## Description
This project is an introduction to Object-Oriented Programming (OOP) in Python. It covers the fundamental concepts of classes, objects, attributes, methods, encapsulation, and properties. The primary goal is to transition from procedural programming to object-oriented design.

## Learning Objectives
By the end of this project, you should be able to explain:
* What is OOP and "first-class everything"
* The difference between a class, an object, and an instance
* What are public, protected, and private attributes
* How to use the `__init__` method and the `self` keyword
* Data Abstraction, Data Encapsulation, and Information Hiding
* How to properly write Getters and Setters using `@property`

## Requirements
* All files will be interpreted/compiled on Ubuntu 20.04 LTS using Python 3 (version 3.8.5).
* All Python scripts must be executable.
* The first line of all files must be `#!/usr/bin/python3`.
* Code must follow the `pycodestyle` (version 2.7.*) guidelines.
* **Documentation is mandatory:** All modules, classes, and methods must have properly formatted docstrings explaining their purpose.
